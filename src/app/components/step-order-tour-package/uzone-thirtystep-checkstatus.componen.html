Cara Pembayaran

    1. Pilih Menu TRANSAKSI LAINNYA pada ATM
    2. Lalu pilih TRANSAKSI PEMBAYARAN
    3. Pilih Jenis Pembayaran TELEPON/HANDPHONE
    4. Masukkan 12 digit Payment Code di atas dengan tepat (021xxx "tanpa 0 di depan)
    5. Periksa kembali Payment Code Anda dan pilih YA untuk melanjutkan pembayaran
    6. Simpan Bukti Pembayaran yang Anda dapatkan
